# Rest my eyes runtime

This folder is reserved for the small visual runtime that will be opened from a floating **Rest my eyes** sticker while readers are inside Ringside View.

Planned interaction:

1. A small floating sticker appears on the reading view.
2. The sticker says `Rest my eyes`.
3. When clicked, the visual runtime opens as a calm overlay or dedicated viewer.
4. The viewer must include an obvious escape control, such as `Return to reading`, so readers can close it and continue Ringside View.

Expected runtime file:

```text
theorem_zero_canonical_substrate_v1_0_native_visual_runtime.html
```

Design notes:

- The runtime should remain visually quiet and optional.
- It should not interrupt or replace the reading path.
- It should honour the canonical substrate / geometry provenance.
- The reading page should keep keyboard and pointer escape paths.
- The first install should be reversible: one button, one overlay, one return control.

Do not delete this README when uploading the runtime. Add the runtime file beside it in this folder.
